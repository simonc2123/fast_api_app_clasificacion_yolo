# Microproyecto 2 — Kubernetes en Azure (AKS)

**Computación en la Nube | Universidad Autónoma de Occidente**

Despliegue de aplicaciones de inteligencia artificial en un clúster de Kubernetes administrado en Azure (AKS). El proyecto incluye una API de clasificación de imágenes, y una aplicación de detección de objetos con YOLOv8 usando FastAPI y Streamlit, ambas orquestadas con Kubernetes y expuestas mediante balanceadores de carga públicos.

## Estructura

- **PUNTO1/** — Clúster AKS en Azure Captura de pantalla como evidencia en la carpeta.
- **PUNTO2/** — App de clasificación de imágenes (Flask + PyTorch ResNet18)
- **PUNTO3/** — App de detección de objetos (FastAPI + YOLOv8 + Streamlit)
- **PUNTO4/** — Supervisión y monitoreo en AKS. Captura de pantalla como evidencia en la carpeta.
